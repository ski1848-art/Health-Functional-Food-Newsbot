2. **필터링** — 모아담다 제품군(건강/다이어트/피부&항산화/모로실/혈당관리) 관련 키워드로 1차 필터링

4. **발송** — 슬랙 채널에 헤더 메시지 + 스레드 형태로 최대 10건 발송

## 슬랙 채널 변경 방법

다른 채널로 발송하고 싶을 때:

1. 슬랙에서 해당 채널에 봇 초대: `/invite @봇이름`
2. GitHub 레포 → Settings → Secrets and variables → Actions
3. `SLACK_CHANNEL_ID` 값을 새 채널 ID로 수정
   - 채널 ID 확인: 슬랙에서 채널 우클릭 → 채널 세부 정보 보기 → 하단 채널 ID 복사
